# 排序演算法實作評量 (Sorting Quiz)

本專案包含兩個網頁工具，用於評量與練習常見的排序演算法（插入排序、選擇排序、氣泡排序）。

## 檔案說明

- **exam.html**: 【考試版】正式評量工具。包含成績自動上傳至 Google 試算表的功能。
- **practice.html**: 【練習版】即時回饋工具。包含排序法說明提示與每輪即時批改功能。

## 技術規格

- 純 HTML/CSS/JavaScript 實作，不需額外後端伺服器（成績上傳除外）。
- 整合 Google Apps Script 進行成績收集。
- 採用響應式設計，支援行動裝置與列印功能。
