選擇排序法（Selection Sort）
每輪從未排序部分找出最小值，移至前方，由小至大
觀念
每輪從未排序的部分掃描一遍，找出最小值，將它與未排序部分的第一個元素交換位置。
第1輪
從全部 5 個找最小值，與第 1 個位置交換。
第2輪
從第 2～5 個找最小值，與第 2 個位置交換。
規律
第 N 輪結束後，前 N 個位置已放入正確的值，不會再被動到。
原始：[5, 3, 8, 1, 2]
第1輪：[1, 3, 8, 5, 2] ← 最小值 1 與 5 交換
第2輪：[1, 2, 8, 5, 3] ← 最小值 2 與 3 交換
第3輪：[1, 2, 3, 5, 8] ← 最小值 3 與 8 交換

插入排序法（Insertion Sort）
每輪將一個元素插入已排序部分的正確位置，由小至大
觀念
將陣列分為「已排序」和「未排序」兩部分。每輪取出未排序的第一個元素，往已排序部分從右往左比較，找到正確位置插入。
第1輪
取第2個元素，與第1個比較，決定是否插入到前面。
第2輪
取第3個元素，與已排序的前2個依序比較，找到正確位置插入。
規律
第 N 輪結束後，前 N+1 個元素已排序完成，其餘不動。
原始：[5, 3, 8, 1, 2]
第1輪：[3, 5, 8, 1, 2] ← 3 插入到 5 前面
第2輪：[3, 5, 8, 1, 2] ← 8 已在正確位置
第3輪：[1, 3, 5, 8, 2] ← 1 插入到最前面
第4輪：[1, 2, 3, 5, 8] ← 2 插入到 1 後面

氣泡排序法（Bubble Sort）
每輪相鄰元素比較並交換，最大值逐漸浮至末端，由小至大
觀念
從陣列頭部開始，相鄰兩個元素比較，若左邊大於右邊就交換，讓大的數字像泡泡一樣逐漸「浮」到末端。
第1輪
比較第(1,2)、(2,3)…(6,7)，共6次比較，最大值移到最後。
第2輪
比較第(1,2)、(2,3)…(5,6)，共5次，第二大值移到倒數第二。
規律
第 N 輪結束後，後 N 個位置已確定，下一輪比較範圍縮短一格。
原始：[5, 3, 8, 1, 2]
第1輪：[3, 5, 1, 2, 8] ← 8 浮到最後
第2輪：[3, 1, 2, 5, 8] ← 5 浮到倒數第二
第3輪：[1, 2, 3, 5, 8] ← 3 浮到正確位置
